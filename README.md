# Terminal Typing Tutor

A port of GNU Typist (GTypist) — improve your touch-typing skills in the terminal.

Forked from [justinsgithub/terminal-typing-tutor](https://github.com/justinsgithub/terminal-typing-tutor).

## Install

```bash
pip install -e .
ttt
```
